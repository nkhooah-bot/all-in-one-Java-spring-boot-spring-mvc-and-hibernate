Spring Boot Configuration Properties

Spring Boot offers over 1,000 configuration properties located in application.properties, enabling environment-specific adjustments without code changes.

1. Web & Server Configuration

Adjust how your server exposes endpoints and manages incoming traffic.

PropertyDescription
server.portChanges the listener port (Default: 8080)
server.servlet.context-pathSets a global prefix for all URLs
server.servlet.session.timeoutSets HTTP session expiry (e.g., 20m)
2. Logging Management

Configure visibility and log storage destinations based on package hierarchy.

# Set specific package logging levels
logging.level.org.springframework=DEBUG
logging.level.org.hibernate=TRACE

# Redirect logs to a file instead of console
logging.file.name=myapp.log
3. Security & Actuator

Override auto-generated defaults for production security and endpoint monitoring.

# Define custom authentication
spring.security.user.name=admin
spring.security.user.password=password

# Change base path for actuator endpoints
management.endpoints.web.base-path=/monitor
4. Data Access (JDBC)

Standard properties for establishing connectivity with backend RDBMS.

spring.datasource.url=jdbc:mysql://localhost:3306/db_name
spring.datasource.username=John
spring.datasource.password=Smith

Note: Always refer to the Official Spring Boot Documentation for the complete list of available properties.
