Setter Injection in Spring Boot

A comprehensive guide to using setter methods for dependency injection.

Implementation Steps
  • 1Create Setter: Define a public setter method in your target class.
  • 2Annotate: Use @Autowired on the setter method to trigger Spring's injection process.
Code Example
@RestController
public class CourseController {
    
    private Instructor myInstructor;

    @Autowired
    public void setInstructor(Instructor instructor) {
        this.myInstructor = instructor;
    }

    @GetMapping("/programmingexercise")
    public String getProgrammingExercise() {
        return myInstructor.getProgrammingExercises();
    }
}
How to Test

1. Run: Start your application. Spring will detect the @Autowired setter and inject the bean.

2. Verify: Visit http://localhost:8080/programmingexercise. Success is confirmed if you see the instructor's returned string.

3. Hot Reload: Use Ctrl+S after changing the code to trigger an automatic context reload via spring-boot-devtools.

Injection Strategy

Choose Constructor Injection for required dependencies to ensure immutability, and Setter Injection for optional dependencies where you need the flexibility to re-assign or omit the dependency.

Success! Your Spring application is correctly wired using setter-based dependency injection.
