Coding Lab: Custom Property Injection

In this hands-on session, we define custom configuration values and bridge them into our REST controller using the @Value annotation.

1. Define Custom Properties

Navigate to src/main/resources/application.properties and add your key-value pairs:

instructor.name=John Smith
student.class=Java Class J02
2. Inject & Expose via REST

Update your SimpleRestController.java to bind these properties to class fields and expose them via a new /classinfo endpoint.

@Value("${instructor.name}")
private String instructorName;

@Value("${student.class}")
private String studentClass;

@GetMapping("/classinfo")
public String getClassInfo() {
  return "Instructor: " + instructorName + ", Class: " + studentClass;
}
3. Execution & Verification
  • 1 Launch: Run your Spring Boot application.
  • 2 Navigate: Access http://localhost:8080/classinfo in your browser.
  • 3 Authenticate: Use your Spring Security credentials (username user and the password from your terminal console).

Browser Output: Instructor: John Smith, Class: Java Class J02
